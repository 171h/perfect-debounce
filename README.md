## Usage

Install package:

```sh
# npm
npm install perfect-debounce

# yarn
yarn install perfect-debounce

# pnpm
pnpm install perfect-debounce
```

Import:

```js
// ESM
import { debounce } from 'perfect-debounce'

// CommonJS
const { debounce } = require('perfect-debounce')
```
